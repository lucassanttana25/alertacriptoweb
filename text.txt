AlertaCryto
  - Nunca mais perca uma oportunidade no volátil mercado de criptomoedas! O CryptoAlerta Inteligente é um sistema robusto e personalizável que notifica você por e-mail quando suas metas de preço de compra ou venda são atingidas, fornecendo estatísticas cruciais para suas decisões.

Visão Geral
  - O AlertaCryto foi desenvolvido para traders e investidores que desejam automatizar o monitoramento de preços de criptomoedas e receber notificações precisas. Chega de ficar checando gráficos o tempo todo! Defina seus preços alvo, e nós te avisamos quando for a hora certa de agir.

Funcionalidades Principais
  - Alertas de Preço Personalizados: Configure facilmente alertas para preços específicos de compra ou venda de suas criptomoedas favoritas.
  - Notificações por E-mail: Receba alertas instantâneos diretamente na sua caixa de entrada assim que suas metas forem alcançadas.
  - Estatísticas Detalhadas da Moeda: Cada notificação inclui dados relevantes da criptomoeda, como variação de preço, volume de negociação e tendências recentes.
  - Relatório de Desempenho do Alerta: Monitore a eficácia dos seus alertas, com informações sobre o tempo de atividade e a frequência de atingimento das metas.
  - Monitoramento em Tempo Real: Conectado diretamente à API do Mercado Bitcoin para dados de preço sempre atualizados.

Tecnologias Utilizadas
Este projeto foi construído com ferramentas modernas e eficientes para garantir performance e escalabilidade:
  - Python: A linguagem principal para o backend do sistema, ideal para processamento de dados e automação.
  - API do Mercado Bitcoin: Fonte de dados confiável e em tempo real para as cotações das criptomoedas.
  - MongoDB: Banco de dados NoSQL flexível e escalável, utilizado para armazenar as configurações dos alertas dos usuários e dados históricos.
